SSH Auto Completion

Dec 23rd, 2013

平时要使用很多服务器，刚用 OS X 的几个月，一直很怀念 SecureCRT，直到习惯了下面的方法。

  1. 在 /etc/hosts 里面定义一些别名，比如 pgi-1 表示 生产环境的 game 应用的 iPhone 版本的第一个节点服务器。

  2. 在 ~/.ssh/config 中准备配置信息，用户名、端口、私钥等等。

  3. Terminal 中使用，ssh pgi-1 连接了，如果配好了公钥验证，那么密码都不需要了。

最近越发的懒了，而且 ZSH 的 Completion List 确实比 Bash 体验好了很多，于是有了下面的脚本。

我把他丢在 ~/.oh-my-zsh/plugins/ssh 下面

然后修改 ~/.zshrc， 在 plugin 设置里面加上 ssh

现在执行 ssh pg<TAB>，就会自动完成了。执行 ssh pg<TAB><TAB> 会列出所有生产环境的游戏服务器。
